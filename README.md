
# Halot Mage Pro Dashboard

CrealityのHalot Mage Proをまとめて管理できるアプリケーションです。  
※このアプリケーションは、Crealityのアプリではありません。Crealityへの問い合わせはご遠慮ください。

## 機能
- LAN内のHalot Mage Proを自動検出
- .cxdlpv4ファイルをHalot Mage Proに送信
- 進捗確認
- 一時停止/再開、停止

![overview](https://mrtska.net/images/08dc34b1-a6d7-439f-8174-043dbfbadc3f)

## インストール

GitHubのリリースページからダウンロードしてください。  
署名はついていません。
